## Build Setup
``` bash
# install dependencies
npm install
# serve with hot reload at localhost:3000
npm run dev
# build for production with minification
npm run build
# build for production and view the bundle analyzer report
npm run build --report
```
For a detailed explanation on how things work, check out the [guide](https://vitejs.dev/guide/) and [docs for rollup](https://rollupjs.org/guide/en/).
